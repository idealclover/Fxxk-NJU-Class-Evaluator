# 南哪大学课程评估自动化脚本

> 📢教务处NB！

本脚本用来自动进行南哪大学的学期末课程评估

默认选项为全部“优秀”，老师表现“较好”，可在源码中进行更改

## 使用方法

首先把**自己想特殊评价的课程**在教务网上评价完成

运行main.py，输入用户名，密码，验证码

等待运行结束

## 部署相关

建议运行环境：Anaconda & Spyder，无需安装其他依赖

如使用python，所用到的库：
* requests
* skimage
* beautifulsoup

> pip3 install scikit-image, requests, beautifulsoup4
